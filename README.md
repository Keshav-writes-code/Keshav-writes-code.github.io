# keshav-writes-code

My personal portfolio and blog — built with [Astro](https://astro.build/), Tailwind CSS, TypeScript, and a touch of SolidJS.

Live at [keshav-writes-code.github.io](https://keshav-writes-code.github.io)

## 💻 Development

All commands are run from the root of the project:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Install dependencies                             |
| `npm run dev`             | Start local dev server at `localhost:4321`        |
| `npm run dev:network`     | Start dev server on local network                |
| `npm run build`           | Build production site to `./dist/`               |
| `npm run preview`         | Preview the build locally before deploying       |
